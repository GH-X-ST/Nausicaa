- Primitive: short LQR-stabilised manoeuvre with entry checks, finite horizon, exit checks, metrics, and failure labels.
- LQR controller: time-invariant local feedback controller embedded inside one primitive, identified by `controller_id`, `linearisation_id`, Q/R weights, gain checksum, and closed-loop audit status.
- Primitive-controller variant: the active library item combining one primitive entry set, reference state/command, finite horizon, local LQR Q/R/K, exit checks, failure labels, and claim status.
- Environment context: local flow and safety features computed from the current state and wind field.
- Episode: one simulated or physical launch from the fixed nominal launch gate.
- R5 W0/W1 robust synthesis: the heavy primitive-library creation and tuning stage. W0 uses dry/near-dry no-updraft environment rows while still applying W3-style plant and implementation perturbations.
